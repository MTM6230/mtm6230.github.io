    # Animation Assignment

    Objective

    For this assignment you will be demonstrating your understanding of Web Animations and the transition and animation properties. You will do this by creating 3 different animation on three separate HTML page (They can share a single CSS file). Each animation must meet their specific requirements below.

    Requirements

    There are 3 parts to this assignment. Each part has it own requirements. All of the requirements must be met in order to complete the assignment successfully.

    Animated Button & Example

    1. Create an animated button that will animate on the :hover and :active states using the transition properties.
    2. The :hover and :active states must be animated differently.
    3. At least 2 properties must be animated during the :hover and :active states animations. (The same two properties maybe used for both)
    4. The 2 properties must use different transitions during the animation sequence.
    5. The following is just an example of what the animate button COULD look like

    Loading Animation & Example

    1. Create a loading animation using an @keyframes at-rule and the animation property. 
    2. The @keyframes at-rule must have at least 3 keyframes.
    3. At least 2 properties must be animated
    4. The animation must repeat forever
    5. The following is just an example of what the loading animation COULD look like

    Animated Sequence & Example

    1. Create an animated sequence
    2. At least 3 HTML elements must be animated during the sequence
    3. At least 3 different @keyframes at-rules must be used.
    4. The sequence must come to an end with the final keyframe being applied to at least 2 of the HTML elements. All elements must cease animating.
